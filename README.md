# project-4
Basic Login Form
Description

A simple username & password login form built with HTML, CSS, and JavaScript.
This project is perfect for beginners learning web development and form validation.

Features

Username and password fields

Basic front-end validation (checks for empty fields)

Responsive and clean design

Easy to integrate with backend authentication

Technologies Used

HTML5 – Structure of the form

CSS3 – Styling the form

JavaScript – Form validation

Installation

Clone the repository:

git clone https://github.com/yourusername/login-form.git

Navigate into the project folder:

cd login-form

Open the HTML file in your browser:

open index.html
Usage

Enter a username

Enter a password

Click Login

If any field is empty, an alert will notify you
